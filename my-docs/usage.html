    Information¶

    Octave¶

    class pymusic.octave.Octave(value)¶

    Contains a value representing the octave of interest.

    There are 9 octaves with 1 corresponding to the lowest octave and 8 to the highest octave. Middle C is found in octave 4. See https://www.w3.org/2021/06/musicxml40/musicxml-reference/elements/octave/ for more information regarding octaves in music XML.

    Parameters:

    value (int) – the current octave.

    pymusic.octave.Octave.__init__(self, value)¶
    Parameters:

    value (int) – current octave

    Raises:

    ValueError – if the octave does not fall within the allowed range.

    Note Name¶

    class pymusic.note_name.NoteName(value)¶

    The names of each note in an octave as defined by ASPN.

    Each octave starts at C.

    pymusic.note_name.get_note_name_from_string(value: str)¶

    Given the note name as a string, returns the matching enum.

    Is case-insensitive and will strip white space.

    Parameters:

    value (str) – the note name to look up

    Returns:

    The NoteName corresponding to the passed string

    Return type:

    NoteName

    Raises:

    ValueError – if the string passed is not a valid note name.
